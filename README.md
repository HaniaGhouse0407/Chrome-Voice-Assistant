## 🧠 Overview

Hands-free browser control via real-time speech recognition. Whisper transcribes audio, spaCy detects intent (search, navigate, play), and Selenium executes the corresponding Chrome action — all within 2 seconds.

---

## ✨ Features

- ✅ Whisper (base/medium) for accurate offline speech transcription
- ✅ spaCy NLP intent classification — search, navigate, play, email
- ✅ Selenium ChromeDriver for browser automation
- ✅ Real-time status indicator (Idle → Listening → Thinking → Executing)
- ✅ Type-a-command fallback mode
- ✅ Session command history with intent log

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/HaniaGhouse0407/Chrome-Voice-Assistant.git
cd Chrome-Voice-Assistant

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set environment variables (if needed)
cp .env.example .env
# Edit .env with your API keys

# 4. Run
streamlit run app.py
```
